# Use Zoho for the business mailbox

The dedicated `kontakt@engelaart.no` inbox will use Zoho Mail Free if custom-domain hosting is available in the selected data center, with Zoho Mail Lite as the paid fallback. Cloudflare manages domain DNS and Resend handles automated transactional sends, while Pi-hosted mail is rejected because deliverability, security, reverse DNS, uptime, and backup risks outweigh the small potential saving.
