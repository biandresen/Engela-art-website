# Use PostHog's free tier for cookieless funnel analytics

V1 will use PostHog Cloud's free tier in the EU region, configured without persistent identity, autocapture, session replay, advertising, or form capture. An explicit event allowlist goes through one small analytics interface so provider details remain localized and the site gains aggregate funnel data without collecting inquiry contents or tracking visitors across sites; Plausible remains a paid fallback if PostHog no longer fits.
