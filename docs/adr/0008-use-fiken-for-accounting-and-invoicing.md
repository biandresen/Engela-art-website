# Use Fiken for accounting and invoicing

Engela Art will trial Fiken as the preferred accounting and invoicing system because it is designed for small Norwegian ENKs, supports compliant invoicing and bookkeeping, and includes multiple user access for the artist and bookkeeping helper. V1 uses the base plan with manual reconciliation and no website/API integration; optional bank, KID, and tax-return features are added only if the trial shows they reduce real work.
